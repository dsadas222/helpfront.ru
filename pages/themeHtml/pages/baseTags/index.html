Начало работы

Перед началом чтения текущей статьи, проверьте, чтобы у вас была установлена visual studio: code и вы работаете на системе windows (от версии xp). Если всё хорошо, то продолжайте пункты списка:

  • Создайте где-то пустую папку (рекомендую на рабочем столе)

  • Запустите vs:code и нажмите в верхнем левом углу на кнопку файл (file) и в открывшемся списке выберите "открыть папку" (open folder). Найдите вашу пустую папку из первого пункта и откройте её

  • В левой колонке создайте файл index.html

  • Создайте базовую разметку в index.html. Для этого напишите просто восклицательный знак (shift + 1) и нажмите Enter

  • Вы готовы к изучению материала

Особенные атрибуты

p

Парный тег для размещения любого текстового контента (абзацы, тексты, слова)

Нет

img

Одинарный тег для добавления изображений на сайт. Ссылку на картинку можно указать прям из интернета или локальный путь

src - путь, где находится изображение

alt - описание изображения

a

Парный тег для добавления ссылки на любой другой файл или сайт в интернете

href - указывает путь на ресурс, который хотите перейти

div

Парный тег для обёртки каких-то элементов, связанных между собой логически

Нет

h1

Парный тег для заголовка. Отличается от обычного <p> размерами и жирностью

Нет

span

Парный тег выделения букв или слова в предложении. Обычно потом к нему добавляют какой-то класс

Нет

Пожелания и советы

  • Всегда старайтесь сразу написать открывающий и закрывающий тег, чтобы не было проблем в расположении элементов

  • Не забывайте про табуляцию (вместо пробелов, ставьте tab). Это помогает понять какой тег находится в другом
