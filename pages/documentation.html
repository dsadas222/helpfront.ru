Заголовки

Всего существует 3 вида заголовков, отличающихся между собой только размерами. Применяется только к тегам "p", а не для h1-h6.

Не надо использовать для изменения размеров текста! Для этого есть модификаторы размера текста

  • titleBig - самый большой размер 32px

  • titleMedium - средний размер 26px

  • titleLittle - самый маленький размер 20px

Модификаторы для таблицы

Если вам не нравится таблица по умолчанию, то вы можете её немного модифицировать на своё усмотрению из заготовленных классов. Для того, чтобы они применились, добавьте их тегу "table":

  • tableHover - убирает у стандартных таблиц голубые полоски и они теперь появляются при наведении на нужную вам строчку

  • tableAllCenter - выравнивает весь текст внутри ячеек по центру (если это возможно)

  • tableFirstTitle - добавляет заголовки в первой ячейки каждого столбца

  • Здесь может быть ваш модификатор, жду предложений для дополнения
